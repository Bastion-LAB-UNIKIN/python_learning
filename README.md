# Bastion-LAB-UNIKIN

Ce dépôt constitue la fondation du programme **Bastion-Lab-coaching-python**.

Il est dédié à l’apprentissage structuré de **Python**, depuis les bases du langage jusqu’aux premiers outils de **data science**.

Le dépôt couvre notamment :

- les fondamentaux du langage Python  
- la programmation scientifique avec **NumPy** et **SciPy**  
- la manipulation de données avec **Pandas**  
- la visualisation avec **Matplotlib**  
- l’introduction au **Machine Learning** avec **Scikit-learn**

L’objectif est de construire une **base solide en programmation et analyse de données**, nécessaire pour la suite du parcours Bastion-Lab.
